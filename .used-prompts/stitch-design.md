Use distinct, attention-grabbing visual indicators (e.g., a glowing border, elevation shadow, or distinct color tag) for tasks that match the user's current location.
> 2. **Add Task Modal:** A clean, intuitive form allowing the user to input the task title, description, and pinpoint a location via a map UI or address search autocomplete. Include a radius slider for location sensitivity.
> 3. **Task Management View:** A chronological and proximity-based sorting of all pending tasks.
